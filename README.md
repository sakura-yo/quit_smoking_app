# 禁煙記録アプリ（Flutter）

スマートフォン向けの禁煙記録アプリです。

## 機能

- **喫煙ボタン**: タップで現在日時を記録（常に表示）
- **喫煙記録タブ**: カレンダーで日付選択、その日の本数・金額表示、月合計、履歴（〇本目・前回からの経過・削除）
- **手入力タブ**: 日付・時刻を選んで過去の記録を追加
- **設定タブ**: 一日の目標本数、ひと箱の値段（1箱＝20本で計算）
- **目標到達**: 記録追加でその日の本数が目標に達したとき、中央にダイアログで通知

## 必要な環境

- Flutter SDK 3.0 以上
- Android / iOS の開発環境（実機またはエミュレータ）

## 実行方法

```bash
cd quit_smoking_app
flutter pub get
flutter run
```

実機・エミュレータを複数接続している場合は `flutter devices` で確認し、`flutter run -d <device_id>` で対象を指定できます。

## データの保存

記録・設定は端末の **SharedPreferences**（Android）/ **UserDefaults**（iOS）に保存されます。アプリ削除でデータも消えます。
